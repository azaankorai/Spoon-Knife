# Backend Engineer — Helio Robotics

**Company:** Helio Robotics
**Location:** Remote (US)
**Posted:** 2026-06-01
**Link:** https://example.com/jobs/helio-robotics-backend-engineer

## About the role

Helio Robotics is looking for a Backend Engineer to help build the services that power
our fleet-management platform. You'll work primarily in Python and PostgreSQL, design
APIs consumed by our mobile and web clients, and help operate the system in production.

## Responsibilities

- Design, build, and maintain REST APIs in Python (FastAPI/Django)
- Improve reliability and observability of backend services running on AWS
- Collaborate with frontend engineers on API contracts and schema design
- Participate in an on-call rotation and write post-incident reviews
- Review code and mentor other engineers on the team

## Requirements

- 4+ years professional experience building backend services
- Strong Python skills; experience with FastAPI, Django, or Flask
- Experience with relational databases (PostgreSQL preferred) and SQL
- Familiarity with AWS or another cloud provider, Docker, and CI/CD pipelines
- Comfortable working in a small team and communicating async (remote-first)

## Nice to have

- Experience with infrastructure-as-code (Terraform/CloudFormation)
- Experience with React or another modern frontend framework
- Prior experience mentoring junior engineers
